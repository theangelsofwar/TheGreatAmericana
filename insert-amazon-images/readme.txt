== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/AmazonImages` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Amazon Images screen to configure the plugin
4. You can get the trial key form the url <a href="https://www.amzimage.com/trial/" target="_blank">https://www.amzimage.com/trial/</a> and These trial key is valid for 7-days.
5. Paste license keys that you get from the <a href="https://www.amzimage.com/" target="_blank">amzimage.com</a> .
6. Next, paste in your Amazon Associates API information, as well as your desired tracking IDs to make sure your affiliate link is placed on each image.
